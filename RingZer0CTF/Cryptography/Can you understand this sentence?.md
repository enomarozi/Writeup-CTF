</b><h3>Solution</h3></b>
<p>Merupakan bubblebabble encoding, decrypt cipher</p>

```python
from bubblepy import BubbleBabble

bb = BubbleBabble()
decod = bb.decode("xipak-comok-repuk-vanik-dytuk-dimyk-sinyx")
print(decod)

```
